# Nigerian Death Incident Analysis

## Project Overview
This project analyzes reported death incidents in Nigeria to identify patterns, trends, and key insights that can support data-driven decision-making. The analysis focuses on understanding incident distribution across locations, time, and categories.

## Tools Used
- Power BI
- Microsoft Excel

## Key Insights
- Geographic distribution of reported death incidents
- Trends over time
- Incident categories and frequency
- High-risk locations and patterns
